## 1. 마시러 가자

- go get/have

    - Let's go have/get some coffee.

- go grab : casual한 뉘앙스로, 편안한 사이에서 주로 사용

## 2. Go + 동사원형

- '~하러 가는', '가서 뭔가를 하는' 자연스러운 흐름이 있는 동사와 사용

    - go have
    - go get
    - go see
    - go meet
    - go buy
    - go eat
    - go find
    - go check
    - go grab
    - go talk to 
    - go ask
    - go play
    - go try
    - go get ready

- 가서 코트좀 가져올게

    - Let me go grab/get my coat.

- 이번 주말에 영화 보러 갈래?

    - Would you like to go see a movie?

- 속상하면 걔한테 가서 얘기를 나눠봐.

    - If you're upset, go talk to her.

- 가서 형이랑 같이 놀아.

    - Go play with your brother.

- 가서 간식 좀 사가지고 올래?

    - Could you go buy some snacks?

- 주말에 할머니 잘 계신지 보고 와야 해.

    - I need to go check on my grandmother this weekend.

- 가서 학교 갈 준비 해.

    - go get ready for school.

## 3. Go + 동사ing

- activity(활동) 하러 갈 때 사용 (레저, 취미, 야외활동 등)

    - go shopping

    - go grocery shopping
    
    - go window shopping

    - go swimming
    
    - go fishing

    - go hiking

    - go camping
    
    - go jogging
    
    - go running

    - go skiing

    - go snowboarding

    - go dancing

    - go bowling

- 먹을 게 다 떨어졌어. 우리 장 보러 가야 해.

    - We're out of food. We need to go grocery shopping.

- 저는 퇴근하고 일주일에 세 번 수영하러 가요.

    - I go swimming three times a week after work.

- 셀리 여기 없어요. 달리러 나갔어요

    - Sally is not here. She went go running.

- 나 이번 주말에 등산 가고 싶어.

    - I'd like to go hiking this weekend.

- 우리 어제 춤추러 나갔었어.

    - We went go dancing yesterday.
